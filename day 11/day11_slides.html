Code that repeats automatically - like playing a song on repeat
List
A container that holds multiple values - like a playlist of songs
Function

Why Loops? ü§î

Without loops (boring and repetitive):

print("Hello") print("Hello") print("Hello") print("Hello") print("Hello")

With loops (smart and efficient):

for i in range(5): print("Hello")
Real Example: Instagram uses loops to show all your photos. Without loops, they'd need separate code for each photo!

For Loops - Count It! üî¢

The for loop repeats code a specific number of times:

# Count from 0 to 4 for i in range(5): print(i) # Output: 0, 1, 2, 3, 4

Different range() options:

# Start at 1, go to 5 for i in range(1, 6): print(i) # 1, 2, 3, 4, 5 # Count by 2s for i in range(0, 10, 2): print(i) # 0, 2, 4, 6, 8

While Loops - Keep Going! üîÅ

While loops repeat as long as a condition is True:

# Password checker password = "python123" guess = "" while guess != password: guess = input("Password: ") if guess != password: print("Wrong! Try again.") print("Access granted! ‚úÖ")

Simple counter with while:

count = 0 while count < 5: print(f"Count is {count}") count = count + 1 print("Done!")
Tip: Be careful! A while loop can run forever if the condition never becomes False!

Lists - Store Multiple Items! üìù

Lists let you store many values in one variable:

# Creating lists friends = ["Alice", "Bob", "Charlie"] scores = [95, 87, 92, 88] mixed = ["Hello", 42, 3.14, True] # Print the whole list print(friends) # Access items by position (index) print(friends[0]) # Alice (first) print(friends[1]) # Bob (second) print(friends[2]) # Charlie (third)

Remember: List positions start at 0, not 1!

First item = index 0, Second item = index 1, etc.

List Operations üõ†Ô∏è

# Start with a list games = ["Minecraft", "Fortnite"] # Add items games.append("Roblox") print(games) # Now has 3 games # Change an item games[1] = "Among Us" print(games) # Get list length size = len(games) print(f"You have {size} games") # Check if item is in list if "Minecraft" in games: print("You have Minecraft!")

Try it! Create your own list of favorite foods and modify it!

Loops + Lists = Power! üí™

Loop through each item in a list:

# List of students students = ["Emma", "Liam", "Olivia", "Noah"] # Print each student for student in students: print(f"Hello, {student}!") # Calculate average grades = [85, 92, 78, 95, 88] total = 0 for grade in grades: total = total + grade average = total / len(grades) print(f"Average: {average}")

Notice: The loop variable (student, grade) takes each value from the list!

Functions - Reusable Code! üîß

Functions are like recipes - write once, use many times!

# Define a function def greet(): print("Hello!") print("Welcome to Python!") # Use (call) the function greet() # Runs the code inside greet() # Can use it again! # Function with a parameter def greet_person(name): print(f"Hello, {name}!") greet_person("Alice") greet_person("Bob")
Real World: Every app button click calls a function! The "like" button on Instagram calls a function that adds a like.

Functions that Return Values üì§

Functions can calculate and send back a result:

# Function that returns a value def add_numbers(a, b): result = a + b return result # Use the function answer = add_numbers(5, 3) print(answer) # 8 # Shorter version def multiply(x, y): return x * y print(multiply(4, 7)) # 28

Practical example:

def calculate_tax(price): tax_rate = 0.08 tax = price * tax_rate return tax item_cost = 25.00 tax = calculate_tax(item_cost) total = item_cost + tax print(f"Total: ${total}")

Common Mistakes to Avoid! ‚ö†Ô∏è

‚ùå Forgetting the colon in loops/functions

# WRONG for i in range(5) print(i) # CORRECT for i in range(5): print(i)

‚ùå List index out of range

nums = [1, 2, 3] # WRONG - no index 3! print(nums[3]) # CORRECT print(nums[2]) # Last item

‚ùå Infinite while loop

# WRONG - never ends! while True: print("Help!") # CORRECT - has an exit count = 0 while count < 5: print(count) count += 1
