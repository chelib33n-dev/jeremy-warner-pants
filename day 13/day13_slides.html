Question 1 of 60
What is a bit?
A) 8 binary digits
B) A single binary digit (0 or 1)
C) A type of computer memory
D) A programming language
Correct Answer: B A bit is the smallest unit of data in computing and represents a single binary digit - either 0 or 1. The word "bit" is short for "binary digit."
Question 2 of 60
What is the decimal value of the binary number 1010?
A) 8
B) 10
C) 12
D) 16
Correct Answer: B Binary 1010 = (1×8) + (0×4) + (1×2) + (0×1) = 8 + 0 + 2 + 0 = 10 in decimal.
Question 3 of 60
How many bits are in one byte?
A) 4
B) 6
C) 8
D) 16
Correct Answer: C A byte is made up of 8 bits. This is a fundamental unit in computing, and all file sizes are ultimately measured in bytes.
Question 4 of 60
Which storage size is the LARGEST?
A) Megabyte (MB)
B) Kilobyte (KB)
C) Terabyte (TB)
D) Gigabyte (GB)
Correct Answer: C The order from smallest to largest is: Byte → KB → MB → GB → TB. A Terabyte is about 1,000 GB or 1 trillion bytes!
