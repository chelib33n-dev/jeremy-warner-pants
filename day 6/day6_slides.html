Vocabulary Time! 📚

Cell
A single box in a spreadsheet where data lives. Like a parking spot for one piece of information. Named by column letter and row number (A1, B2, etc.)
Formula
A calculation that starts with = sign. Like giving the spreadsheet math homework that it solves instantly. Example: =A1+B1
Function
A pre-built formula that does common tasks. Like shortcuts on your phone - instead of typing a whole address, you just tap "Home". Example: =SUM(A1:A10)

More Vocabulary! 📖

Range
A group of cells selected together. Like highlighting multiple messages to delete at once. Written as A1:B5 (from top-left to bottom-right)
Cell Reference
Using a cell's address in a formula. Like tagging a friend in a post - you're pointing to their content. When you type =A1*2, you're referencing cell A1

Basic Formulas 🧮

Every formula starts with = to tell the spreadsheet "Hey, calculate this!"

=A1+B1 → Adds two cells
=A2-B2 → Subtracts B2 from A2
=C3*D3 → Multiplies (use * not x)
=E4/F4 → Divides E4 by F4
Pro tip: Click on cells instead of typing their addresses - less mistakes!

Power Functions 💪

Functions are like cheat codes for common calculations:

=SUM(A1:A10) → Adds all cells from A1 to A10
=AVERAGE(B1:B20) → Finds the average (mean)
=MAX(C1:C100) → Finds the biggest number
=MIN(D1:D50) → Finds the smallest number
=COUNT(E:E) → Counts how many numbers in column E

Pro Tips from Brother Cope 🎯

Tip 1: Double-click between column headers to auto-resize columns
Tip 2: Use Ctrl+C and Ctrl+V to copy formulas to other cells
Tip 3: Drag the little square in the corner of a cell to copy it down
Tip 4: Press Enter to move down, Tab to move right
Tip 5: Format as currency: Select cells → Format → Number → Currency

Common Rookie Mistakes 🚫

Don't worry, we all make these at first!

  • ❌ Forgetting the = sign (nothing happens!)
  • ❌ Using x instead of * for multiply
  • ❌ Typing cell names wrong (=A11 instead of =A1)
  • ❌ Wrong cell references (always double-check!)
  • ❌ Deleting cells that formulas need (#REF! error)
Remember: Undo is your friend! Ctrl+Z saves the day!
