Copying Formulas & Cell Anchors ‚öì

When you copy formulas, cell references change automatically... unless you anchor them!

Regular References (Move when copied)

=A1+B1

Copy down ‚Üí becomes =A2+B2, then =A3+B3, etc.

Anchored References (Stay fixed with $)

=$A$1+B1

Copy down ‚Üí becomes =$A$1+B2, then =$A$1+B3

The $ signs "lock" that part of the reference!

Types of Anchors:

  • $A$1 - Both column AND row locked (never changes)
  • $A1 - Column locked, row can change
  • A$1 - Row locked, column can change

Pro Tip: Press F4 (or Fn+F4) while editing a formula to cycle through anchor types!

Power Combos! üöÄ

The real magic happens when you combine formulas:

Example 1: Round an Average

Example 2: IF with LEN

Checks if text is too long

Example 3: VLOOKUP with IF
