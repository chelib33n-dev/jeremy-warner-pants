Cell References: The Basics 📍

What are Cell References?

When you use a formula like =A1+B1, the A1 and B1 are cell references - they point to specific cells!

Two Types of References:

  • Relative References (like A1) - Change when you copy them
  • Absolute References (like $A$1) - Stay the same when copied

Think of it like directions:

Relative: "Turn left at the next street" (changes based on where you are)

Absolute: "Go to 123 Main Street" (same address no matter where you start)

💡 Remember: Most of the time you want relative references, but sometimes you need absolute!

Relative References (Default) 🔄

How Relative References Work

Cell C1: =A1+B1

Copy to C2: Becomes =A2+B2

Copy to C3: Becomes =A3+B3

Example: Calculating Totals

A B C
10 5 =A1+B1 (shows 15)
20 8 =A2+B2 (shows 28)
30 12 =A3+B3 (shows 42)

Key Point: The formula adjusts automatically! Each row adds its own numbers.

Absolute References: The Dollar Signs! 💲

Locking Cells with $

Dollar signs "lock" parts of your reference so they don't change when copied!

$A$1 = Both column (A) and row (1) are locked

$A1 = Only column is locked

A$1 = Only row is locked

Example: Sales Tax Calculation

Tax Rate in cell E1: 0.08 (8%)

A B C
Item Price Tax
Shirt $20 =B2*$E$1
Pants $35 =B3*$E$1

💡 Pro Tip: Press F4 (Windows) or Cmd+T (Mac) to quickly add $ signs to a reference!

Task 2: Field Trip Fee Formula 💰

The Challenge:

Everyone pays the same field trip fee (shown in cell M2: $15.00)

In cell F3, type: =$M$2

Why the dollar signs?

$M$2 is an absolute reference - it won't change when we copy the formula down!

After typing the formula:

  3. Copy (Ctrl+C or Cmd+C)
  4. Select F4 through the last student
  5. Paste (Ctrl+V or Cmd+V)

Task 3: VLOOKUP for Shirt Prices 👕

How VLOOKUP Works:

=VLOOKUP(what_to_find, where_to_look, which_column, exact_match)

Our Lookup Table (Data sheet)

SizePrice
S$8.00
M$9.00
L$10.00
XL$12.00

Think of it like: "Look up the size in column E, find it in the Data sheet, tell me the price from column 2"

Writing the VLOOKUP Formula 🔍

In cell G3, type exactly:

=VLOOKUP(E3,Data!A:B,2,FALSE)

Breaking it down:

  • E3 = The shirt size to look up
  • Data!A:B = Look in columns A and B of the Data sheet
  • 2 = Return the value from the 2nd column (Price)
  • FALSE = Find an exact match only

After entering: Copy this formula down to all students. Each should show the correct price for their size!

Common Mistake: Forgetting "Data!" before A:B. This tells Sheets to look in the Data sheet!

Task 4: Calculate Total Due 💵

Task 5: IF Statement for Actions 📧

The Logic:

IF the form is not received → Email Parents
ELSE → OK

In cell J3, type:

=IF(I3="No","Email Parents","OK")

How it reads:

  • IF cell I3 equals "No"
  • THEN show "Email Parents"
  • ELSE show "OK"

Copy down! You should see "Email Parents" for students with "No" in Form Received, and "OK" for those with "Yes"

Real World: Schools use IF statements to automatically flag missing permissions, unpaid fees, and incomplete registrations!

Common Problems & Solutions 🔧

VLOOKUP shows #N/A?

  • Check spelling of shirt sizes (must match exactly!)
  • Make sure you typed Data!A:B not just A:B
  • FALSE must be spelled correctly

Formulas not copying down?

  • Select the cell with the formula
  • Copy (Ctrl+C)
  • Select ALL cells where you want it
  • Paste (Ctrl+V)

Summary functions showing 0 or error?

  • Check your range (A3:A50 etc.)
  • Make sure you're counting the right column
