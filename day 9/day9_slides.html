Working with Text (Strings) üìù

You can combine text using the + sign:

first_name = "Spider"
last_name = "Man"
superhero = first_name + last_name
print(superhero) # Output: SpiderMan

# With a space
superhero = first_name + " " + last_name
print(superhero) # Output: Spider Man

Getting Input from Users üé§

Make your programs interactive with input():

name = input("What's your name? ")
print("Nice to meet you, " + name + "!")

age = input("How old are you? ")
print("Wow,", age, "is a great age!")

The input() function:

  • Stops and waits for the user to type something
  • Saves what they type into a variable
  • Like asking a question and waiting for an answer!

Comments - Notes for Humans üìå

Comments help explain your code (computer ignores them):

# This is a comment - Python ignores this line
print("Hello") # This prints Hello

# You can write multiple lines
# of comments to explain
# complicated stuff

"""
You can also use triple quotes
for multi-line comments
like this!
"""

Use comments to:

  • Explain what your code does
  • Leave notes for yourself
  • Temporarily disable code
